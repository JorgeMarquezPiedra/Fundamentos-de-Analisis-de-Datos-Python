# Fundamentos-de-Analisis-de-Datos-Python

**Lenguaje: Python 3.x**

**Librerías: Pandas, Matplotlib, Seaborn, NumPy**

**Entorno: Google Colab / Jupyter Notebook**

## Este repositorio consiste en un proyecto dedicado a la experimentación técnica y el fortalecimiento de habilidades en Análisis de Datos. Aquí documento ejercicios prácticos de limpieza de datos, análisis estadístico y entrenamiento de modelos predictivos de regresión lineal.
